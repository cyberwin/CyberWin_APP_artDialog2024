# 未来之窗web网页弹窗对话框

#### 介绍
网页弹窗，独立使用单文件版本，可指定位置，左上，左下，中间，右侧，下册，左侧，右侧，顶部，底部。支持拖动开关，标题显示开关，支持同时多个弹窗，支持遮罩和非遮罩，弹窗支持透明度
弹窗支持大小，高度设定
关闭按钮：可选
适用于混合开发app框架，家用电视，商用智能盒子显示
窗口ID可以指定，便于独立销毁和移动放大
